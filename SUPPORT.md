## ❓ Frequently Asked Questions

### General Questions

**Q: Is Omarflix free?**  
A: Yes! Omarflix is free and open-source under the MIT license.

**Q: Do I need to create an account?**  
A: No. Everything works locally on your device without any accounts.

**Q: Is my data private?**  
A: Yes, 100%. All data stays on your device. No cloud uploads unless you explicitly enable sync.

**Q: Does it work offline?**  
A: Yes! Once videos are loaded, everything works offline. Posters require internet on first load only.

### Setup Questions

**Q: Where do I download Omarflix?**  
A: Check the GitHub Releases page for the latest `omarflix.html` file.

**Q: Do I need to install anything?**  
A: No. Just save the HTML file and open it in your browser.

**Q: What browsers are supported?**  
A: Chrome 90+, Firefox 88+, Safari 14+, Edge, Opera, and any modern browser.

### Video & Format Questions

**Q: What video formats are supported?**  
A: MP4, MKV, WebM, AVI, MOV, and M4V (H.264 recommended).

**Q: My video is in a format Omarflix doesn't support. What do I do?**  
A: Convert it using FFmpeg, Handbrake, or VidCoder to MP4 (H.264 + AAC).

**Q: Why doesn't my video play?**  
A: Check browser console (F12) for errors. Your browser may not support the codec. Try converting to MP4.

### Watch History & Progress

**Q: Where is my watch history stored?**  
A: In your browser's `localStorage`. It's on your device, not in the cloud.

**Q: Will my progress sync between devices?**  
A: Only if you enable sync and export/import manually. See README.md for cross-device syncing.

**Q: How do I clear my watch history?**  
A: Click "Clear History" in the player toolbar or menu.

**Q: Can I backup my watch history?**  
A: Yes. Click the Sync button (⟲) → Export Settings to save as JSON.

### Mobile Questions

**Q: Can I install Omarflix as an app on my phone?**  
A: Yes! Open in Safari (iOS) or Chrome (Android) → "Add to Home Screen" or "Install app".

**Q: Why is Omarflix slow on my phone?**  
A: Try with fewer videos, close other apps, or refresh the page. Make sure you have 2GB+ RAM.

### Troubleshooting

**Q: I can't add folders on Safari/iOS, what do I do?**  
A: iOS has limited directory picker support. Try selecting files individually instead.

**Q: My library doesn't load automatically, how do I fix it?**  
A: Check browser privacy settings allow `localStorage`. Don't use Incognito mode for persistence.

**Q: Posters aren't showing up. Is that normal?**  
A: Yes. First load may take 30+ seconds to fetch posters. Refresh after 1 minute. Emoji placeholders are normal.

**Q: Videos play audio only, no video. Help?**  
A: Your browser doesn't support the video codec. Convert to MP4 (H.264).

## 🐛 Found a Bug?

1. **Check existing issues** — Search GitHub Issues to see if it's already reported
2. **Collect information:**
   - Browser name and version
   - Operating system
   - Video file format
   - Error message (from F12 console)
   - Steps to reproduce
3. **Open an issue** — Use the Bug Report template
4. **Provide details** — The more information, the faster we can help

👉 [Report a Bug](https://github.com/username/omarflix/issues/new?template=bug_report.md)

## 📞 Getting Help

### Before Asking

1. Read the [README](README.md) — Most questions are answered there
2. Check [CHANGELOG](CHANGELOG.md) — See what changed
3. Search [existing issues](https://github.com/username/omarflix/issues) — You might not be the first
4. Check browser console (F12) — Copy any error messages

### How to Ask

**Good issue example:**
```
Browser: Chrome 125
OS: Windows 11
Problem: Video plays audio only, no video showing
Steps: Open MP4 file, click play
Error: (paste from F12 console)
```

**Poor issue example:**
```
doesn't work
help pls
```

## 🆘 Code Issues?

If you need help debugging code:

1. Open browser console (F12)
2. Look for red error messages
3. Copy the full error
4. Share in issue with steps to reproduce
5. Include browser version
